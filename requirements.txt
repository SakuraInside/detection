# Integra Native — Rust-only runtime.
#
# Python в РАНТАЙМЕ не используется (см. README). Все три процесса (video-bridge,
# backend_gateway, integra_ffi) — нативный код. Сам launcher `run.py` написан
# на чистой stdlib и НЕ требует pip-зависимостей.
#
# Pip-deps здесь оставлены ПУСТЫМИ намеренно: если они когда-нибудь снова
# потребуются (тулинг, экспорт .onnx → .engine, бенчмарки), добавьте сюда.
#
# Для запуска достаточно:
#   • Python >= 3.10 (только для run.py)
#   • Rust toolchain (cargo)
#   • CMake + C++17 (для integra_ffi)
#   • OpenCV (для video-bridge: opencv-rust требует libclang при сборке)
#   • CUDA + TensorRT — опционально, для GPU inference (Linux рекомендуется)
